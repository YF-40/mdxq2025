MosaicMate README

画像の一部を、ぼかし・モザイク・黒塗り・スタンプなどで加工するローカルツール

1. 概要

MosaicMate は、画像の一部を手軽に隠したり、強調したりするためのローカル画像加工ツールです。

HTML / CSS / JavaScript の 1ファイル完結アプリとして動作します。個人情報、顔、メールアドレス、住所、画面キャプチャ内の不要部分などを加工できます。

2. 主な特徴

ブラウザだけで動作し、インストール不要
画像はローカルで処理
画像選択、D&D、Ctrl+V貼り付けに対応
範囲指定をマウス操作と数値入力で調整可能
Ctrl + Z で元に戻す
Ctrl + ドラッグ / Ctrl + ホイールで表示操作

3. 対応している加工モード

加工モード説明主な用途
ぼかし指定範囲をぼかします顔、氏名、住所などを自然に隠す
モザイク指定範囲をブロック状に荒くします顔、個人情報、画面の一部を隠す
黒塗り指定範囲を黒で塗りつぶします確実に読めないようにする
白塗り指定範囲を白で塗りつぶします書類や画面キャプチャの不要部分削除
半透明塗り指定範囲を半透明色で覆います隠しつつ背景を少し残す
枠線指定範囲に枠線を描きます注目箇所の強調
斜線指定範囲に斜線を描きますマスク済み表現、注意喚起
グレースケール指定範囲を白黒化します重要でない部分を弱める
ノイズ指定範囲にノイズを加えます内容を読みにくくする
絵文字スタンプ絵文字をスタンプとして配置します顔隠し、注釈、目印
顔文字スタンプ顔文字をスタンプとして配置します軽い雰囲気の顔隠し、注釈

4. 使い方

4.1 画像を読み込む

  1. 「画像を選択 または D&D」をクリックして画像を選択
  2. 画像ファイルをドラッグ&ドロップ
  3. クリップボードに画像をコピーして Ctrl + V
  4. 「クリップボードから貼付」ボタンをクリック

4.2 範囲加工

  1. 加工モードを選択します。
  2. 画像上で加工したい範囲をドラッグします。
  3. 必要に応じて、X / Y / 幅 / 高さの数値を修正します。
  4. 効果ごとのスライダーを調整します。
  5. 「選択範囲に適用」をクリックします。

4.3 スタンプ

  1. 「絵文字スタンプ」または「顔文字スタンプ」を選択します。
  2. プリセットからスタンプを選ぶか、入力欄に好きな文字を入力します。
  3. スタンプサイズ、回転、透明度を調整します。
  4. 画像上をクリックすると、その位置にスタンプが配置されます。

Windows では Win + . で絵文字入力パネルを開けます。

5. 便利な操作

操作内容
Ctrl + Z1つ前の加工に戻す
Command + ZMac系ブラウザで1つ前の加工に戻す

表示サイズ調整画像全体が見えるように表示倍率を調整
PNG保存加工後の画像をPNG形式で保存

6. 画面構成

場所説明
画面上部アプリ名を表示します
左側：操作パネル画像読み込み、加工モード、範囲指定、各種設定を操作します
右側：画像表示エリア読み込んだ画像を表示し、範囲選択やスタンプ配置を行います
右上ツールバー拡大縮小、100%表示、表示サイズ調整を行います

7. ファイル構成

MosaicMate は基本的に以下の1ファイルで動作します。

mosaicmate_commented.html

このHTMLファイルの中に、HTML / CSS / JavaScript がすべて含まれています。

8. 修正しやすいポイント

HTML

画面の場所ごとにコメントを入れています。

<!-- HTML：画像読み込みエリア -->
<!-- HTML：加工モード選択 -->
<!-- HTML：画像表示エリア -->

CSS

どの画面部品の見た目を担当しているかをコメントしています。

/* CSS：画面上部のアプリ名ヘッダー */
/* CSS：左側の操作パネル */
/* CSS：画像表示エリア */

JavaScript

機能ごとにコメントを入れています。

// 画像ファイル選択、ドラッグ&ドロップ、Ctrl+V貼り付けに対応します。
// マウスで選択した範囲を数値入力欄に反映します。
// Ctrl+Z / Cmd+Z で「元に戻す」を実行します。

9. 注意事項

  • 画像処理はブラウザ上で行います。
  • 大きすぎる画像では、ブラウザの動作が重くなる場合があります。
  • クリップボード画像の読み取りは、ブラウザや権限設定によって動作が異なる場合があります。
  • 保存形式はPNGです。
  • 完全な匿名化が必要な場合は、ぼかしやモザイクよりも黒塗りの方が安全です。

10. 推奨ブラウザ

  • Google Chrome
  • Microsoft Edge

Clipboard APIを使う機能は、ブラウザやページの開き方によって制限される場合があります。Ctrl + V による貼り付けは比較的動作しやすいです。
