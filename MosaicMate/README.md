# MosaicMate

**MosaicMate** は、画像の一部を手軽に隠したり、強調したりするためのローカル画像加工ツールです。  
HTML / CSS / JavaScript の **1ファイル完結アプリ**として動作します。

個人情報、顔、メールアドレス、住所、画面キャプチャ内の不要部分などを、ぼかし・モザイク・黒塗り・スタンプなどで加工できます。

---

## 1. 主な特徴

- ブラウザだけで動作
- インストール不要
- 画像はローカルで処理
- PNG / JPG / WebP などの画像に対応
- 画像選択、ドラッグ&ドロップ、クリップボード貼り付けに対応
- 範囲指定をマウス操作と数値入力の両方で調整可能
- Ctrl + Z で元に戻す
- Ctrl + マウスドラッグで画像表示位置を移動
- Ctrl + マウスホイールで拡大・縮小
- 加工後の画像を PNG として保存可能

---

## 2. 対応している加工モード

| 加工モード | 説明 | 主な用途 |
|---|---|---|
| ぼかし | 指定範囲をぼかします | 顔、氏名、住所などを自然に隠す |
| モザイク | 指定範囲をブロック状に荒くします | 顔、個人情報、画面の一部を隠す |
| 黒塗り | 指定範囲を黒で塗りつぶします | 確実に読めないようにする |
| 白塗り | 指定範囲を白で塗りつぶします | 書類や画面キャプチャの不要部分削除 |
| 半透明塗り | 指定範囲を半透明色で覆います | 隠しつつ背景を少し残す |
| 枠線 | 指定範囲に枠線を描きます | 注目箇所の強調 |
| 斜線 | 指定範囲に斜線を描きます | マスク済み表現、注意喚起 |
| グレースケール | 指定範囲を白黒化します | 重要でない部分を弱める |
| ノイズ | 指定範囲にノイズを加えます | 内容を読みにくくする |
| 絵文字スタンプ | 絵文字をスタンプとして配置します | 顔隠し、注釈、目印 |
| 顔文字スタンプ | 顔文字をスタンプとして配置します | 軽い雰囲気の顔隠し、注釈 |

## 3. 使い方

### 3.1 画像を読み込む

以下のいずれかで画像を読み込みます。

1. 「画像を選択 または D&D」をクリックして画像を選択
2. 画像ファイルをドラッグ&ドロップ
3. クリップボードに画像をコピーして **Ctrl + V**
4. 「クリップボードから貼付」ボタンをクリック

---

### 3.2 ぼかし・モザイクなどの範囲加工

1. 「加工モード」を選択します。
2. 画像上で加工したい範囲をドラッグします。
3. 必要に応じて、X / Y / 幅 / 高さの数値を修正します。
4. 効果ごとのスライダーを調整します。
5. 「選択範囲に適用」をクリックします。

マウスで範囲を選択すると、数値入力欄にも自動反映されます。  
数値を変更すると、選択範囲とプレビューも更新されます。

---

### 3.3 絵文字スタンプ・顔文字スタンプ

1. 加工モードで「絵文字スタンプ」または「顔文字スタンプ」を選択します。
2. プリセットからスタンプを選ぶか、入力欄に好きな文字を入力します。
3. スタンプサイズ、回転、透明度を調整します。
4. 画像上をクリックすると、その位置にスタンプが配置されます。

Windows では **Win + .** で絵文字入力パネルを開けます。

---

## 4. 便利な操作

| 操作 | 内容 |
|---|---|
| Ctrl + Z | 1つ前の加工に戻す |
| Command + Z | Mac系ブラウザで1つ前の加工に戻す |
| Ctrl + マウスドラッグ | 画像表示位置を移動 |
| Ctrl + マウスホイール | 画像を拡大・縮小 |
| 100% | 画像を100%表示に戻し、画像左上を表示部左上に合わせる |
| 表示サイズ調整 | 画像全体が見えるように表示倍率を調整 |
| PNG保存 | 加工後の画像をPNG形式で保存 |

---

## 5. 画面構成

| 場所 | 説明 |
|---|---|
| 画面上部 | アプリ名を表示します |
| 左側：操作パネル | 画像読み込み、加工モード、範囲指定、各種設定を操作します |
| 右側：画像表示エリア | 読み込んだ画像を表示し、範囲選択やスタンプ配置を行います |
| 右上ツールバー | 拡大縮小、100%表示、表示サイズ調整を行います |

## 6. ファイル構成

MosaicMate は基本的に以下の1ファイルで動作します。

```text
mosaicmate_commented.html
```

このHTMLファイルの中に、HTML / CSS / JavaScript がすべて含まれています。

---

## 8. 注意事項

- 画像処理はブラウザ上で行います。
- 大きすぎる画像では、ブラウザの動作が重くなる場合があります。
- クリップボード画像の読み取りは、ブラウザや権限設定によって動作が異なる場合があります。
- 保存形式はPNGです。
- 完全な匿名化が必要な場合は、ぼかしやモザイクよりも黒塗りの方が安全です。
